Load project

Weather application

My Weather Application was initally designed with InVision Freehand. After developping a basic wireframe, I developed a prototype in the Editor. Thanks to a succesful OpenWeatherMap API integration, the app displays real-time data of various cities in the world. You can easily enter your preffered location in the search bar on top. However, by simply clicking on the location button, data will be shown of your current location.

Tools: InVision Freehand, HTML, CSS, JS, API, Responsive Web Design
